# Job Management System

A simple backend-style Java project that simulates a Job Management System using Object-Oriented Programming concepts and CRUD operations.

## Features

- Add Job
- View All Jobs
- Search Job By ID
- Search Job By Name
- Update Job
- Delete Job

## Technologies Used

- Java
- ArrayList
- OOP Concepts

## Concepts Practiced

- Classes and Objects
- Encapsulation
- Constructors
- Getters and Setters
- Method Overriding (`toString()`)
- CRUD Operations
- Service Layer Architecture

## Project Structure

```text
Job.java          -> Model class for job data
JobService.java   -> Handles business logic and CRUD operations
Main.java         -> Runs and tests the application
